[2.1]

Pre-Game Commands

Before entering the game proper, you see the pre-game menu. The available commands are:

KeyAction
EEnter the game. Takes you to the main game menu.
BRead the bulletin. Displays the sysop's bulletin board posting.
VVillage stats. Shows information about the current village.
IInstructions. Displays the game help file.
TToday's news. Read the news from today's game events.
YYesterday's news. Read the news from the previous day.
SSee scores. Displays the clan rankings for this village.
GGame settings. Shows the current game configuration.
LLeague scores. Displays rankings across all villages in the InterBBS league (only available if InterBBS is enabled).
HHelp. Displays the general help screen.
QQuit. Exit The Clans and return to the BBS.
