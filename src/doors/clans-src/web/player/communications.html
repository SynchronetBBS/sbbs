15.1Overview 15.2Mail 15.3Public Messages and Chat
[15.1]

Overview

The communications menu lets you send and receive messages. Press C from the main menu to access it:

KeyAction
CCheck mail. Read private messages sent to your clan.
WWrite mail. Send a private message to another clan.
PPublic mail. Post to the public message board.
UUpdate last read. Set your last-read message pointer.
GGlobal message. Post a message visible across all villages in the InterBBS league (only available if InterBBS is enabled).
VView stats.
QReturn to the main menu.
[15.2]

Mail

Private mail is delivered between clans on the same village. Use "Write mail" to compose a message and choose a recipient from the clan list. Use "Check mail" to read messages others have sent you.

The "Update last read" option lets you mark all current messages as read so that only new messages appear next time.

[15.3]

Public Messages and Chat

Public mail is posted to a message board visible to all players in the village. Use this for announcements, challenges, or general discussion.

Chat (/) is available in most locations -- the street, mines, market, church, training hall, and town hall. Each location has its own chat room.

Note: You can chat up to 4 times per day across all locations.
